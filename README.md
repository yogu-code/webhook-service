# Webhook Delivery System

A production-ready **webhook delivery platform** designed to reliably send events from one system to another with guaranteed delivery semantics.

Built using modern technologies and best practices, this system handles incoming events, queues them, delivers them securely, and manages retries intelligently — solving one of the most critical challenges in event-driven architectures.

## ✨ Highlights

- **Reliable Delivery Engine** powered by **BullMQ + Redis**
- **Smart Retry Mechanism** with configurable exponential backoff
- **Secure Webhooks** using HMAC signatures
- **Complete Delivery Observability** with detailed logging and analytics
- **User Management** with JWT authentication
- **RESTful APIs** for event ingestion, subscriptions, and delivery logs

## 🛠️ Tech Stack

- **Framework**: NestJS (TypeScript)
- **Queue System**: BullMQ + Redis
- **Database**: PostgreSQL with Prisma ORM
- **Authentication**: JWT
- **Security**: HMAC Signature Validation
- **Others**: Class Validator

## 🎯 Purpose

This project was developed to gain deep, hands-on experience with background job processing, worker architecture, distributed queuing systems, and building resilient event delivery infrastructure.
